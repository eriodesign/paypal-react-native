# paypal-react-native

RN library using Paypal Checkout SDK

## Installation

```sh
npm install paypal-react-native
```

## Usage

```js
import { PaypalReactNativeView } from "paypal-react-native";

// ...

<PaypalReactNativeView color="tomato" />
```
